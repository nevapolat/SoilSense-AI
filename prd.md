## 1. Project Overview
SoilSense AI is a web application for small-scale producers providing regenerative agriculture guidance and smart composting recipes.

## 2. Core Features (MVP)
- **Dashboard:** Real-time weather and estimated soil health for the user's location.
- **AI Compost Wizard:** A form where users input available waste, and Gemini provides N-P-K balanced compost recipes.
- **Regenerative Advisor:** Chemical-free, restorative soil solutions powered by Gemini API.
- **Alert System:** Early warnings for extreme weather (frost, heavy rain).

## 3. Tech Stack
- React, Tailwind CSS, Lucide Icons, Gemini Pro API.
